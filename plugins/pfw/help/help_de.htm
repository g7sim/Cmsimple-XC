Pfw_XH – Handbuch

Pfw_XH ist ein Plugin-Framework für CMSimple_XH, das als CMSimple_XH Plugin verpackt ist.

Wichtiger Hinweis

Pfw_XH konnte noch nicht in diversen Umgebungen getestet werden, und seine API ist noch nicht stabil. Es handelt sich um eine so genannte Alpha-Version. Vorsicht bei der Verwendung!

Voraussetzungen

Pfw_XH benötigt CMSimple_XH ≥ 1.6.3 und PHP ≥ 5.4.0.

Installation

Die Installation erfolgt wie bei vielen anderen CMSimple_XH-Plugins auch. Im CMSimple_XH Wiki finden sie ausführliche Hinweise.

  1. Sichern Sie die Daten auf Ihrem Server.
  2. Entpacken Sie die ZIP-Datei auf Ihrem Computer.
  3. Laden Sie das gesamte Verzeichnis pfw/ auf Ihren Server in das plugins/ Verzeichnis von CMSimple_XH hoch.
  4. Vergeben Sie Schreibrechte für die Unterverzeichnisse css/ und languages/.
  5. Navigieren Sie nach Plugins → Pfw und prüfen Sie, ob alle Voraussetzungen für den Betrieb erfüllt sind.

Einstellungen

Die Konfiguration des Plugins erfolgt wie bei vielen anderen CMSimple_XH-Plugins auch im Administrationsbereich der Homepage. Wählen Sie Plugins → Pfw.

Die Lokalisierung wird unter Sprache vorgenommen. Sie können die Zeichenketten in Ihre eigene Sprache übersetzen (falls keine entsprechende Sprachdatei zur Verfügung steht), oder sie entsprechend Ihren Anforderungen anpassen. Siehe auch Native Language Support.

Das Aussehen von Pfw_XH kann unter Stylesheet angepasst werden.

Verwendung

Dieses Plugin hat keine eigene Funktionalität. Es dient als Framework und Toolkit für andere Plugins. Dokumentation für Pluginentwickler findet sich im SDK. Übersetzer sollten das Kapitel über Native Language Support lesen.
