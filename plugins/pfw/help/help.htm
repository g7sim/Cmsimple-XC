Pfw_XH – User Manual

Pfw_XH is a plugin framework for CMSimple_XH packaged as CMSimple_XH plugin.

Important Notice

Pfw_XH hasn't been tested so far in various environments, and its API is not stable yet. It is a so called alpha version. Use with caution!

Requirements

Pfw_XH requires CMSimple_XH ≥ 1.6.3 and PHP ≥ 5.4.0.

Installation

The installation is done as with many other CMSimple_XH plugins. See the CMSimple_XH wiki for further details.

  1. Backup the data on your server.
  2. Unzip the distribution on your computer.
  3. Upload the whole directory pfw/ to your server into CMSimple_XH's plugins/ directory.
  4. Set write permissions for the subdirectories css/ and languages/.
  5. Browse Plugins → Pfw and check if all requirements are fulfilled.

Settings

The plugin's configuration is done as with many other CMSimple_XH plugins in the website's back-end. Select Plugins → Pfw.

Localization is done under Language. You can translate the character strings to your own language, if there is no appropriate language file available, or customize them according to your needs. See also Native Language Support.

The look of Pfw_XH can be customized under Stylesheet.

Usage

This plugin doesn't have any own functionality. It serves as framework and toolkit for other plugins. Documentation for plugin developers can be found in the SDK. Translators should read the the chapter on Native Language Support.
