Plib_XH

Plib_XH is a utility plugin serving other plugins. It is always hidden; that means it is not displayed under Plugins. You can only check whether it is installed via Settings→Info, or the Hi_updatecheck plugin.

Requirements

Plib_XH is a plugin for CMSimple_XH. It requires CMSimple_XH ≥ 1.7.0 and PHP ≥ 7.1.0 with the JSON extension.

Download

The lastest release is available for download on Github.

Installation

The installation is done as with many other CMSimple_XH plugins.

  1. Backup the data on your server.
  2. Unzip the distribution on your computer.
  3. Upload the whole directory plib/ to your server into the plugins/ directory of CMSimple_XH.

Settings

Plib_XH does not provide any configuration, language settings or a stylesheet.

Developer Usage

See https://cmb69.github.io/Plib_XH/.
